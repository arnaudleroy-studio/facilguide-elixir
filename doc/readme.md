# FacilGuide Elixir Utilities

Multilingual tech guide utilities for Elixir by [Facil.guide](https://facil.guide). Supports 5 languages.

## Installation

Add `facilguide` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:facilguide, "~> 0.1.0"}
  ]
end
```

## Usage

```elixir
FacilGuide.version()
# => "0.1.0"

FacilGuide.base_url()
# => "https://facil.guide"
```
